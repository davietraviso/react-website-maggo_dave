# Maggo
Maggo adalah platform C2B2C berbasis web yang menawarkan solusi inovatif untuk pengelolaan sampah organik melalui penjualan dan penjemputan sampah dari rumah tangga dan bisnis, yang secara efisien disalurkan ke peternak maggot. 

## Fitur Maggo

- **Jasa**: Layanan penjemputan limbah organik dari pengguna.  
- **Produk**: Marketplace untuk hasil olahan maggot.  
- **Blog**: Artikel edukasi terkait pengelolaan limbah organik.  
- **Ruang Tanya**: Forum diskusi dan konsultasi bagi pengguna.  

## Teknologi  
Aplikasi ini dibangun menggunakan:  

### Development Tools  
- [Visual Studio Code](https://code.visualstudio.com/)  : 
Editor kode sumber gratis dan lintas platform yang dirancang untuk pengembangan perangkat lunak. VS Code mendukung berbagai bahasa pemrograman, ekstensi, dan fitur seperti debugging, Git bawaan, dan IntelliSense.

### Design Tools  
- [Figma](https://www.figma.com/) : 
Alat desain berbasis web untuk membuat prototipe, wireframe, dan desain UI/UX secara kolaboratif. Figma memungkinkan tim bekerja secara real-time pada satu proyek desain.

### Framework dan Libraries  
- [React](https://react.dev/)  :
Pustaka JavaScript yang digunakan untuk membangun antarmuka pengguna (UI). React memungkinkan pengembangan aplikasi web yang cepat dengan konsep komponen dan state management.
- [Vite](https://vite.dev/)  :
Build tool yang cepat dan modern untuk proyek web. Vite mendukung pengembangan aplikasi dengan hot module replacement (HMR) dan waktu build yang sangat cepat, cocok untuk proyek React.
- [Bootstrap](https://react-bootstrap.netlify.app/):
Framework front-end open-source yang digunakan untuk membangun antarmuka web yang responsif dan modern. Bootstrap menyediakan komponen seperti tombol, navigasi, form, dan grid yang siap digunakan.

### Languages  
- [HTML](https://www.w3schools.com/html/html_computercode_elements.asp) :
Bahasa standar untuk membuat struktur halaman web. HTML digunakan untuk menandai elemen-elemen seperti teks, gambar, dan tautan di dalam dokumen web.
- [CSS](https://www.w3schools.com/css/) :
Bahasa yang digunakan untuk mendesain tampilan halaman web. CSS mengatur elemen seperti warna, font, tata letak, dan responsivitas untuk membuat halaman terlihat menarik.
